Key Features
Simple Mediation
X → M → Y with full path coefficients (b and β), Sobel test, indirect effect CIs, and interactive path diagram.
Serial Mediation
X → M1 → M2 → Y with specific indirect paths, total indirect effects, and bootstrapped CIs.
Moderation
X × W → Y with both unstandardized and standardized coefficients in every table. Simple slopes at ±1 SD and mean.
Moderated Mediation
PROCESS-consistent b-path (Y ~ X + M only). Conditional indirect effects, index of moderated mediation.
Annotated Slopes Plot
Beautiful simple slopes plot with coefficient and p-value annotations on each line. Toggle p-values on/off. Full screen + hi-res download.

Statistical Methodology

MedModr uses Ordinary Least Squares (OLS) regression via matrix algebra. Every table reports both unstandardized (b) and standardized (β) coefficients for all predictors.

β = (X'X)⁻¹X'y  |  SE(β) = √[MSE · diag(X'X)⁻¹]  |  t = b/SE  |  CI = b ± t*(df,α/2) × SE
  • Unstandardized b: Raw regression coefficient. Interpreted in original units.
  • Standardized β: Computed by running the identical regression on z-scored variables. Allows comparison of effect sizes across predictors.
  • Moderated Mediation b-path: Y regressed on X and M only (not W) — consistent with PROCESS Model 7.
  • Indirect effects: Product of path coefficients (a×b). Bootstrap CIs use the percentile method.
  • Simple Slopes Plot: Lines show the predicted effect of X on Y (or M) at W = Mean ± 1SD. Each line carries an annotation showing the slope coefficient (b) and its significance; p-values can be toggled on/off. Annotations are placed along the upper portion of each line, clear of the level labels in the legend.
Open Source & Contributing

MedModr is a free, open-source project and is actively maintained. Researchers, statisticians, and developers are welcome to fork the repository, contribute improvements, or adapt the tool for their own purposes.

Get Involved

Whether you want to add a new analysis type, improve the visualisation engine, fix a bug, or translate the interface, contributions of all sizes are appreciated. The entire codebase is a single self-contained HTML file — no build tools required.

Feature ideas, bug reports, and discussion are all welcome via the GitHub Issues tracker.
  • Adding analysis types: Each analysis type is a self-contained run*() function. Follow the pattern of runSimpleMed() to add a new model.
  • Improving the plot engine: The Canvas-based slope renderer lives in drawSlopes(). All visual parameters are accessible at the top of the function.
  • Bug reports: Please include your browser version, the analysis type, and a minimal reproducible dataset.

About This Tool

MedModr is a free web application for mediation and moderation analysis. All statistical computations run entirely in your browser — no data is sent to any server, ever.

  • No login required — use it directly in your browser
  • Privacy-first — all computation is client-side
  • Both b and β — every regression table shows unstandardized and standardized coefficients side by side
  • Annotated simple slopes — coefficients and p-values on each slope line; toggle p-values on/off
  • Automatic interactions — mean-centering and X×W are handled behind the scenes
  • PROCESS-consistent moderated mediation — b-path is Y ~ X + M only

About MedModr

Comprehensive browser-based software for mediation, moderation, and conditional process analysis

Overview

MedModr (Mediation & Moderation) is a free, open-source web application for conducting simple mediation, serial mediation, moderation, and moderated mediation (conditional process) analyses.

The tool implements Ordinary Least Squares (OLS) regression via matrix algebra, following the methodological framework of Hayes (2022) in Introduction to Mediation, Moderation, and Conditional Process Analysis. It provides both unstandardized (b) and standardized (β) coefficients, bootstrap confidence intervals for indirect effects, interactive path diagrams, annotated simple slopes plots, and reporting.

Privacy Guarantee: All data processing occurs locally in your browser. Your data never leaves your device, and no results are sent to any server. You can disconnect from the internet after the page loads and still run analyses.
Scientific Q&A
Are the calculations accurate compared to other established software (e.g., PROCESS macro)?
Yes. MedModr uses standard OLS matrix algebra (X'X⁻¹X'y) identical to that used in other software. Coefficient estimates match within rounding error (±0.001). Bootstrap confidence intervals use percentile method. For example, the tool has been cross-validated against PROCESS macro v4.2 outputs.
What are the minimum sample size recommendations?
The tool requires at least 5 observations to run regressions. For reliable estimates, we recommend: Simple mediation: n ≥ 50 (or 20+ with strong effects); Serial mediation: n ≥ 100; Moderation: n ≥ 80; Moderated mediation: n ≥ 200. Bootstrap CIs (5,000+ samples) are strongly recommended for indirect effects regardless of sample size.
How does MedModr handle covariates (control variables)?
Users can specify an unlimited number of covariates. Each covariate is automatically mean-centered and included in all regression models (a-path, b-path, and total effect models). The first row of the covariate input must contain variable names, followed by data rows with one value per column. Covariates control for their effects on the mediator(s) and outcome, providing more accurate estimates of focal relationships.
What is the Index of Moderated Mediation and when should I use it?
The Index of Moderated Mediation (IMM = a₃ × b) quantifies the linear change in the indirect effect as a function of the moderator W. A bootstrap confidence interval that excludes zero indicates statistically significant moderated mediation (Hayes, 2015). MedModr automatically computes the IMM and provides bootstrap CIs when bootstrap samples are specified.
Does MedModr handle missing data?
Currently, MedModr requires complete data (listwise deletion). All variables must have equal N with no missing values. Users should pre-process their data (e.g., using mean imputation, multiple imputation, or removing cases with missing values) before pasting into MedModr. Future versions may include built-in missing data handling.
What assumptions does MedModr make about the data?
MedModr assumes OLS assumptions: (1) linearity between predictors and outcome, (2) independence of errors, (3) homoscedasticity (constant variance of residuals), (4) normality of residuals (for small-sample inference), and (5) no perfect multicollinearity. The tool does not automatically test these assumptions; users should verify them using external software. Bootstrap CIs are robust to violations of normality.
Does MedModr automatically handle categorical variables?
Manual dummy coding required. MedModr does not automatically convert categorical variables (e.g., gender, treatment group, education level) into dummy codes. All categorical predictors including moderators, and covariates must be converted by the user into numeric 0/1 dummy variables before pasting data into the tool. For a categorical variable with k levels, create k−1 dummy variables. The reference group should be coded as 0 on all dummies. Continuous variables can be used as-is and will be automatically mean-centered when appropriate.
